# Ôn Tập Triết - App Qua Môn

Ứng dụng giúp bạn ôn luyện trắc nghiệm các môn học lý thuyết trên trường một cách hiệu quả nhất.

## Truy cập ứng dụng
Bạn có thể sử dụng ứng dụng trực tuyến tại: [natelyt12.github.io/ontaptriet](https://natelyt12.github.io/ontaptriet)

## Tính năng nổi bật
- Chế độ Tập trung chống xao nhãng.
- Lưu trữ lịch sử, theo dõi tiến độ.
- Giao diện tối ưu có cả Sáng và Tối (Mac OS Style).
- Hỗ trợ đa dạng các môn học và chương học.

## Tác giả
- **Phúc Thanh** (DCCNTT-16.3)
- Built with Gemini AI
